SDE Engine

Shared Dependency Execution (SDE)

The SDE Engine is the core scheduling and dependency management backbone of Gamma Arena. It is designed to handle non-overlapping model dependency trees, ensuring that complex scientific workflows can be parallelized without race conditions on the truth plane. SDE primitives are defined in the Genesis Tree.

Non-overlapping Branches

SDE isolates model dependencies into discrete branches, allowing multiple agents to evolve independent hypotheses simultaneously.

Lazy Reconciliation

State is only reconciled when dependencies converge, reducing the compute overhead of continuous truth publication.

Technical Specifications

The SDE backbone operates as a typed blackboard system where each dependency node is tagged with its mechanistic provenance.

  • Dependency Locking: Prevents multiple models from claiming the same biophysical parameter space within a single turn.
  • Laminar Batching: Groups scientific events into coherent batches for efficient truth-plane publication.
  • Delayed Consolidation: Allows speculative execution to happen in the foreground while truth-plane commits happen asynchronously.
